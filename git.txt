git add :added to the staging zone

git commit -m "Message": commits the changes made

git status: cheks the status of each file weather they are in the staging zone or not

git colne "link from git": clones the repo from your git


#branching

git branch somename: creates a branch with somename

git merge somename: merges the somename branch with current branch

git log: dispalays all the changes commited along with hashcodes

git checkout hashcode: u will move to  that particular branch of hashcode



#adding git details(config)

git config --global user.email "email used in git" : to add git account email

git config --global user.name "user name of git" : to add git account username

git config --global --edit : to change the git account details
